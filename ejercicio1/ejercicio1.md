# Breve explicación escrita de JVM, JDK y JRE

    → JVM: La Java Virtual Machine es una máquina virtual que interpreta el bytecode generado por el compilador (javac) y lo ejecuta de la misma manera independientemente del CPU.

    → JRE: Java Runtime Environment es el entorno en el que se ejecutan los programas Java, incluye dentro de sí a la JVM y las bibliotecas estandar que los programas necesitan para funcionar.

    → JDK: Java Development Kit es el conjunto de herramientas con el que se desarrollan programas Java. Incluye el compilador javac y al JRE, que permite ejecutar los programas.
