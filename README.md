## Compilar y ejecutar desde línea de comandos

Este proyecto utiliza Java JDK 25 y Maven.

### Ruta correcta

Asegurarse de estar en: C:\Users\agust\Documents\Facultad\Sem III\Algoritmos y Estructuras de Datos\Ejercicios\RepartidoUT0 y ejecutar:

cd ejercicioN*
* Debe reemplazar 'N' con el número del ejercicio para que se ejecute correctamente (Ej: cd ejercicio1 para ubicarse en el directorio correcto)

### Compilar el proyecto

Desde la ruta correcta del proyecto ejecutar:

mvn compile

### Ejecutar el programa

Luego ejecutar:

java -cp target/classes com.repartidout0.App

Esto ejecutará la clase principal `App`, que contiene el método `main`.
